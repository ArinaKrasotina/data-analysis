# Pandas + Python: сверка СБИС и аптек

Учебный кейс по обработке бухгалтерских данных:
- загрузка CSV-файлов
- объединение данных через pandas merge
- сверка дат и сумм
- формирование Excel-отчётов

## Стек
- Python 3.10+
- pandas
- openpyxl

## Запуск
```bash
pip install -r requirements.txt
