# Анализ продаж и бонусной программы сети аптек

## Цель проекта
Разработка комплексного дашборда для мониторинга ключевых показателей сети аптек и оценки эффективности бонусной программы лояльности.

## Структура дашборда

### Лист 1. Обзорный дашборд
**Ключевые метрики:**
- Выручка
- Маржинальность
- Количество чеков

**Анализ:**
- Динамика продаж
- Рейтинг аптек
- Структура платежей

### Лист 2. Товарная аналитика
- ABC-анализ товарного ассортимента
- Распределение по ценовым сегментам
- Оценка ликвидности товаров
- Оценка прибыльности товаров

### Лист 3. Анализ бонусной системы
- Оценка активности держателей карт
- Динамика начислений бонусов
- Динамика списаний бонусов
- Сегментация клиентов
- Сравнение чеков участников программы и обычных покупателей

## Основные результаты
- Выявлена прямая корреляция между участием в бонусной программе и ростом среднего чека
- Определены товарные категории, требующие оптимизации ассортиментной политики
- Проведена сегментация клиентской базы для дальнейшей разработки персонализированных предложений

## Техническая реализация
- Инструмент: Power BI
- Источник данных: база Apteka
- Количество DAX-мер: 20+
